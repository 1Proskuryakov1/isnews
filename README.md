# isnews

Интеллектуальный сервис для классификации новостных публикаций.

Проект разрабатывается как выпускная квалификационная работа и строится по двум связанным контурам:

1. Обучение моделей в Google Colab через Jupyter Notebook (`.ipynb`).
2. Демонстрационное приложение на Python для загрузки данных, обучения, сохранения моделей и классификации новостей.

## Выбранный стек

- Python 3.11+
- Streamlit для графического интерфейса
- pandas и numpy для обработки данных
- scikit-learn для базовых моделей и пайплайнов
- PyTorch и transformers для нейросетевых моделей
- joblib и safetensors для сохранения артефактов

## Быстрый запуск

1. Создать виртуальное окружение.
2. Установить зависимости из `requirements.txt`.
3. Запустить приложение командой:

```bash
python main.py
```

На текущем этапе откроется стартовый интерфейс с описанием проекта и будущих модулей системы.
