# Дорожная карта разработки на 50 коммитов

Ниже приведен план итеративной разработки проекта. Каждый пункт можно делать отдельным небольшим коммитом от имени студента.

## Этап 1. Инициализация проекта

1. Создать базовую структуру репозитория, `main.py`, `requirements.txt`, `README.md`.
2. Добавить базовый GUI-каркас приложения.
3. Создать каталоги `data/`, `models/`, `notebooks/`, `reports/`.
4. Добавить модуль конфигурации путей и общих настроек.
5. Подготовить шаблон Colab-ноутбука для обучения.

## Этап 2. Загрузка и хранение данных

6. Реализовать загрузку CSV-файлов из локальной файловой системы.
7. Реализовать загрузку датасета по прямой ссылке.
8. Добавить проверку структуры датасета и обязательных колонок.
9. Реализовать сохранение загруженных датасетов в каталог проекта.
10. Добавить отображение сводной информации о датасете в GUI.

## Этап 3. Предобработка текстов

11. Создать модуль очистки текстов.
12. Добавить нормализацию регистра, пробелов и базовой пунктуации.
13. Реализовать удаление пустых и дублирующихся записей.
14. Добавить анализ длины текстов и распределения классов.
15. Добавить сохранение результатов предобработки.

## Этап 4. Исследовательский анализ данных

16. Реализовать таблицу статистик по датасету.
17. Добавить график распределения классов.
18. Добавить график длины текстов.
19. Реализовать просмотр примеров новостей по выбранному классу.
20. Добавить выгрузку отчета по анализу данных.

## Этап 5. Базовая ML-модель

21. Реализовать разбиение данных на train, validation, test.
22. Создать TF-IDF векторизацию текстов.
23. Добавить обучение Logistic Regression.
24. Реализовать расчет Accuracy, Precision, Recall, F1-score.
25. Добавить сохранение базовой модели и векторизатора.

## Этап 6. Применение базовой модели

26. Реализовать загрузку сохраненной ML-модели.
27. Добавить классификацию одиночного текста.
28. Реализовать пакетную классификацию из файла.
29. Добавить сохранение предсказаний в CSV.
30. Реализовать вкладку сравнения истинных и предсказанных меток.

## Этап 7. Нейросетевая модель

31. Подготовить Colab-ноутбук для обучения нейросети.
32. Реализовать токенизацию текстов для transformer-модели.
33. Добавить обучение компактной русскоязычной модели.
34. Реализовать сохранение нейросетевой модели в каталог `models/`.
35. Добавить логирование истории обучения и метрик.

## Этап 8. Инференс нейросети

36. Реализовать загрузку нейросетевой модели из файла.
37. Добавить классификацию новости нейросетевой моделью через GUI.
38. Реализовать сравнение предсказаний ML- и DL-моделей.
39. Добавить отображение вероятностей классов.
40. Реализовать сохранение результатов экспериментов в отчет.

## Этап 9. Сервисные возможности

41. Добавить единый менеджер артефактов проекта.
42. Реализовать журналирование операций приложения.
43. Добавить валидацию пользовательских параметров обучения.
44. Реализовать настройки гиперпараметров через GUI.
45. Добавить автоматическую проверку наличия файлов модели и данных.

## Этап 10. Завершение проекта

46. Подготовить окончательный `ipynb` для Google Colab.
47. Добавить экспорт метрик и графиков в каталог `reports/`.
48. Подготовить демонстрационный набор моделей `model1`, `model2`.
49. Обновить README и инструкции по воспроизведению результатов.
50. Провести итоговую проверку структуры репозитория и сценариев запуска.
